# d3rabi - Arabic CloudStream Plugins

Arabic streaming plugins for CloudStream.

## Available Plugins

| Plugin | Description | Website |
|--------|-------------|---------|
| **Prestige** (برستيج) | Arabic movies and series | hp.brstej.com |
| **CimaLight** (سيما لايت) | Arabic and foreign movies/series | cimalite.cam |

## Installation

1. Open **CloudStream** app
2. Go to **Settings** → **Extensions** → **Add Repository**
3. Enter:
   ```
   https://raw.githubusercontent.com/mrmakrider/d3rabi/builds/repo.json
   ```
4. Install desired plugins

## Building

```bash
./gradlew make
```
